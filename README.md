# Customer Segmentation Analysis

An end-to-end Customer Segmentation project using Python and Power BI to analyze customer purchasing behavior and identify high-value customer groups using RFM (Recency, Frequency, Monetary) analysis.

---

## Project Overview

This project focuses on segmenting customers based on their transaction history to help businesses better understand customer behavior and improve decision-making in areas such as:
- Customer retention
- Personalized marketing
- Revenue optimization
- Loyalty analysis

Using RFM analysis, customers were grouped into segments such as:
- Best Customers
- Loyal Customers
- At Risk Customers
- Regular Customers

## Methodology

### 1. Data Cleaning
- Removed missing values
- Filtered cancelled transactions
- Created total transaction value column

### 2. RFM Analysis
Calculated:
- **Recency** → Days since last purchase
- **Frequency** → Number of purchases
- **Monetary** → Total amount spent

### 3. Customer Segmentation
Customers were categorized into different groups based on RFM scores.

### 4. Dashboard Development
Built an interactive Power BI dashboard to visualize:
- Customer segment distribution
- Revenue contribution by segment
- Spending behavior
- Customer activity trends

---

## Key Insights

- High-value customers contributed the majority of revenue
- At-risk customers showed low purchase frequency and recency
- Loyal customers demonstrated consistent purchasing behavior
- Customer segmentation can support targeted business and marketing strategies
